## рџ“һ Troubleshooting

### Icon not showing?
- Check asset file exists in `images/` folder
- Verify file path is correct
- Check browser console for 404 errors

### Icon misaligned?
- Adjust `vertical-align` or `margin` values
- Use `style="position:relative;top:2px;"` if needed

### Icon wrong color?
- SVG files inherit theme colors automatically
- PNG files may need `filter: invert(1);` for light mode
- Use `opacity` or `filter: brightness()` to adjust
